# NAME

Minion::Backend::Mango - Mango backend for Minion

# SYNOPSIS

  use Minion::Backend::Mango;

  my $backend = Minion::Backend::Mango->new('mongodb://127.0.0.1:27017');

# DESCRIPTION

Minion::Backend::Mango is highly scalable Mango backend for Minion.
